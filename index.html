datepicker by chrishulbert

SImple, light Javascript date picker, with no javascript framework dependencies.

Dependencies

None

Install

See the demo page - you'll need a js file, a CSS file, a couple lines in your html's head section, and add the 'datepicker' class to your textboxes.
