Date Picker Example

Enter Date / Time:

How to use these pickers in your app:

Put the following 2 or 3 lines in your page's <head> section:

<link rel="stylesheet" type="text/css" href="datepicker.css"> 
<script type="text/javascript" src="datepicker.js"></script>
<script type="text/javascript" src="timepicker.js"></script>

On any <input> that you want to have a date picker for:

  • set/add the 'datepicker' class
  • set an id that will match the id in the onclick function call that follows
  • add a <span> tag with the class of "datepickershow" and an onclick attribute to call showDatePicker() with the id of the input
<input id="date_input" class="datepicker" size='11' title='D-MMM-YYYY'>
<span class="datepickershow" onclick="return showDatePicker('date_input')"></span>

Use the <span> tag to create the calendar icon, best done with an icon font and CSS techniques. A subset of Open Iconic font is included with the package and used in this example.

On any <input> that you want to have a time picker for, set/add the 'timepicker' class:

<input id='start_dt' class='timepicker'>
<input id='another_dt' class='myclass timepicker'>

Enjoy!
